Konzept einer Cloudlösung (TBA)

Das Ziel der Arbeit ist die Konzeption einer Cloudlösung zur Analyse von Daten für eine präventive Wartung in der Automobilbranche.

Pressenvermessungtool (2.0)

Das Ziel der Arbeit ist die Entwicklung von einem Softwarevergleichstool für die Pressenvermessung zur Analyse von Genauigkeitskennwerten.

Zwei Behälter ohne Interaktion (1.7)

Das Ziel der Arbeit ist es zwei Flüssigkeitsbehälter ohne Interaktion als mathematisches Modell in MatLab zu modellieren und diese zu vergleichen.

Bildbearbeitung und -korrektur (1.0)

Das Ziel der Arbeit ist es gegebene Bilder mithilfe von Matlab zu manipulieren bzw. korrigieren. Mit Fokus auf Verarbeitung, Auswertung und Kompression.

Bauweise von Robotern(1.3)

Das Ziel der Arbeit ist es den Unterschied der Bauweise von zwei Industrieroboter auf deren Eigenschaften unter gegebenen Aspekten zu untersuchen.

NC-Programmierung (1.7)

Das Ziel der Arbeit ist die CNC-Programmierung von Fräsmaschinen und die Roboterprogrammierung eines Industrieroboter in einer Roboterzelle.

Machine Learning(1.3)

Das Ziel der Arbeit ist es als Team mit PM-Methoden einen Algorithmus zur Identifikation von Ziffern unter Verwendung von Python zu entwickeln.

Softwareentwicklung (1.7)

Das Ziel der Arbeit ist es ein Petri-Netz-Simulator zu entwickeln und zu implementieren. Mit Fokus auf GUI, Simulator und graphische Auswertung.

Wetterstation mit DHT11 Sensor (1.7)

Das Ziel der Arbeit ist es mit dem Raspberry und einem DHT11 Sensor eine IoT Wetterstation zu entwickeln und in Echtzeit auf beliebigen Geräten abzurufen.

Piezo-Sensoren (2.3)

Das Ziel der Arbeit es es die Anwendung von Piezo-Sensoren in der Prozessmesstechnik zu untersuchen und Unterschiede festzustellen.

Steuerung von Bohreinrichtung (1.3)

Das Ziel der Arbeit ist die Steuerung für das Spannen und Bohren von Werkstücken mit SPS zu entwickeln und in CoDeSys zu simulieren.

Regelungstechnik (1.0)

Das Ziel der Arbeit ist es eine Einführung in Simulink, anhand vorgegebener Aufgaben zu erarbeiten. Z.B. die Parametrierung und Simulation eines MFDS.

Messtechnik (2.7)

Das Ziel der Arbeit ist es verschiedene Messungen mit Multimeter, Oszilloskop zu untersuchen. Es folgt die Modellierung und Simulation von Anwendungen in LabVIEW.

Produktentwicklungsprozess (1.0)

Das Ziel der Arbeit ist ein technischer Bericht zur Herstellung von Kosmetikprodukten und die Konstruktion einer Abfülleinrichtung nach VDI 2221.

Werkstoffkunde (3.0)

Das Ziel der Arbeit ist es Versuche der mechanischen Werkstoffprüfung, Metallographie und Kunststoffe zu analysieren und auszuwerten.

Schlüsselqualifikation (1.0)

Das Ziel der Arbeit ist es ein technisches System weiterzuentwickeln, mit dem Fokus auf die Aspekte die für den Studiengang Mechatronik relevant sind.
